## 🚀 部署步骤

### 1. 准备工作
```bash
# 1. 注册微信小程序账号
# 2. 开通微信云开发
# 3. 安装微信开发者工具
```

### 2. 导入项目
```bash
# 1. 打开微信开发者工具
# 2. 选择 "导入项目"
# 3. 选择项目根目录: D:\AI-Project\AI-DotCircle
# 4. 填写 AppID（或使用测试号）
# 5. 勾选 "使用云开发"
```

### 3. 配置云开发
```bash
# 1. 点击 "云开发" 按钮开通环境
# 2. 记录环境 ID
# 3. 确保云函数使用 DYNAMIC_CURRENT_ENV
```

### 4. 部署云函数
在微信开发者工具中右键点击以下目录，选择 "创建并部署：云端安装依赖"：
- `cloudfunctions/login`
- `cloudfunctions/packages`
- `cloudfunctions/help`
- `cloudfunctions/user`
- `cloudfunctions/credit`
- `cloudfunctions/leaderboard`

### 5. 创建数据库
在云开发控制台数据库中创建：
- `users`
- `packages`
- `helpRecords`
- `creditHistory`

### 6. 运行预览
点击 "预览" 按钮，使用微信扫码查看效果。
